# Quickstart: Dashboard Layout

## Prerequisites
- Node.js 18+
- pnpm

## Running the Dashboard

1.  **Navigate to Frontend**:
    ```bash
    cd frontend
    ```

2.  **Install Dependencies** (if new):
    ```bash
    pnpm install
    ```

3.  **Start Dev Server**:
    ```bash
    pnpm dev
    ```

4.  **View Dashboard**:
    - Open `http://localhost:8080` (or the port shown in terminal).
    - The main landing page is the Dashboard.
    - **Toggle Navigation**: Use the floating button (Top-left on mobile, Bottom-left on desktop).
    - **Resize Window**: Verify layout adaptation (Chart 21:9, Cards 3-col -> Stacked).

## Mocking Data
Currently, the components use internal placeholders or mock props. No backend connection is strictly required for layout verification.
