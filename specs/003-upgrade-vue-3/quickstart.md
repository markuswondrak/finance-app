# Quickstart Guide: Vue 3 Frontend

## Prerequisites

- Node.js 18+ (Required for Vite)
- npm 9+

## Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Development

Start the development server (configured to port 8081):

```bash
npm run dev
```

*Note: The command has changed from `npm run serve` (Vue CLI) to `npm run dev` (Vite).*

## Building

Build for production:

```bash
npm run build
```

## Testing

Run unit tests (Vitest):

```bash
npm run test
```
*Note: This replaces `npm run test:unit`.*
