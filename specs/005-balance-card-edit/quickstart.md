# Quickstart: Current Balance Card

## Prerequisites
- Backend running (`go run ./cmd/server`)
- Frontend running (`pnpm dev`)
- Database migrated (automatic on backend start)

## Usage
1. Open dashboard.
2. Locate "Current Balance" card (top row, left).
3. Click card to open edit modal.
4. Enter new amount.
5. Click Save.
6. Verify chart updates.
